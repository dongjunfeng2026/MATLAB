# LCC-S 无线电能传输（WPT）仿真模型 —— 中文标注说明

> 文件：`LCC_S_2018b.slx`
> 拓扑：原边 LCC 补偿 + 副边 S（串联）补偿
> 工作场景：磁耦合谐振式无线充电系统（IPT/WPT）

---

## 1. 系统总体结构

整个仿真按照能量流方向，可分为下面 6 大部分：

```
 ┌──────────┐  ┌─────────────┐  ┌─────────────────┐  ┌──────────┐  ┌──────────────┐  ┌──────────┐
 │ 直流输入  │→│ 高频全桥逆变  │→│ 原边 LCC 补偿网络 │→│ 耦合线圈  │→│ 副边 S 补偿+整流 │→│ 滤波负载  │
 │  Vin     │  │  Q1~Q4      │  │ Lf  Cf  Cp      │  │ Lp ─ Ls  │  │ Cs + 二极管桥   │  │ Cd + RL  │
 └──────────┘  └─────────────┘  └─────────────────┘  └──────────┘  └──────────────┘  └──────────┘
                                          电气主电路（功率回路）
                                                 │
                                                 ▼
                                       测量、PWM驱动、效率运算
```

---

## 2. 元件 ↔ 中文名称 详细对照表

### 2.1 电源与逆变部分（原边输入级）

| 模型中名称 | 中文名称 | 作用说明 | 关键参数 |
|---|---|---|---|
| `DC Voltage Source1` | **直流输入电源 Uin（Vin）** | 给整个原边提供直流母线电压 | 幅值 = 15 V |
| `Mosfet4` (Q1) | **逆变开关管 Q1**（左桥臂上管） | 全桥逆变的 4 个 MOSFET 之一 | Ron=0.044 Ω |
| `Mosfet5` (Q2) | **逆变开关管 Q2**（右桥臂上管） | 同上，与 Q1 互补 | 同上 |
| `Mosfet6` (Q3) | **逆变开关管 Q3**（左桥臂下管） | 同 Q4 一组，对角导通 | 同上 |
| `Mosfet7` (Q4) | **逆变开关管 Q4**（右桥臂下管） | 与 Q1 同时导通 | 同上 |
| `Pulse Generator` | **PWM 驱动信号 1**（Q1/Q4） | 产生方波驱动对角开关管 | 高频方波（约 85 kHz 级别） |
| `Pulse Generator1` | **PWM 驱动信号 2**（Q2/Q3） | 与 1 反相，互补驱动 | 同上 |
| `Goto/From: Q1,Q2,Q3,Q4` | **PWM 驱动信号传递标签** | 把脉冲信号无线送到对应 MOSFET 的栅极 | — |

> 这 4 个 MOSFET 加上方波信号一起就是**高频全桥逆变器**，作用是把直流变成高频交流方波，驱动后面的 LCC 谐振网络。

---

### 2.2 原边 LCC 补偿网络（一次侧谐振网络）

LCC 补偿即由一个串联电感 + 一个并联电容 + 一个串联电容组成的 T 型网络，用于实现恒流源特性、抗负载和耦合系数变化。

| 模型中名称 | 中文名称 | 拓扑位置 | 参数 |
|---|---|---|---|
| `Series RLC Branch3` | **原边谐振电感 Lf**（LCC 中的 L） | 桥臂中点串联进入 LCC 网络 | L = 4.7 µH，R = 1 Ω（寄生） |
| `Series RLC Branch8`（黄色） | **原边并联补偿电容 Cf**（LCC 中的 C₁/Cp1） | 在 Lf 后面对地并联 | C = 538.94 nF |
| `Series RLC Branch10`（品红色） | **原边串联补偿电容 Cp**（LCC 中的 C₂/Cp2） | 与发射线圈 Lp 串联 | C = 100.12 nF |

> LCC 网络的核心特征：在谐振频率下，发射线圈电流 ILp 与负载几乎无关 → **恒流源**特性，是无线充电常用的拓扑。

---

### 2.3 耦合线圈（磁路部分）

| 模型中名称 | 中文名称 | 说明 |
|---|---|---|
| `Mutual Inductance1`（粉红色） | **发射/接收耦合线圈（Lp - Ls）** | 用 Simscape 的 Mutual Inductance 模块表示原副边两个线圈之间的互感与自感关系 |
|  | 自感 Lp ≈ 30 µH，自感 Ls ≈ 14 µH | 互感 M ≈ 5 µH（耦合系数由几何位置决定） |

> 这是**电能无线传输的"无线"环节**——通过磁场把能量从原边线圈耦合到副边线圈。

---

### 2.4 副边 S 补偿 + 整流部分（二次侧）

| 模型中名称 | 中文名称 | 作用 | 参数 |
|---|---|---|---|
| `Series RLC Branch5`（蓝色） | **副边串联补偿电容 Cs**（即 LCC-**S** 中的 S） | 与接收线圈 Ls 串联，构成副边串联谐振 | C = 180.93 nF |
| `Universal Bridge1`（橙色） | **副边全桥整流器** | 把高频交流转回直流，2 桥臂、二极管器件 | 二极管，Ron=1 mΩ |
| `Series RLC Branch6` | **输出滤波电容 Cd（Co）** | 平滑整流后的直流，减小纹波 | C = 470 µF |
| `Series RLC Branch4` | **负载电阻 RL** | 模拟电池/恒阻负载 | R = 10 Ω |

> 副边只用一颗 **Cs**（Series 串联电容）来补偿接收线圈的感抗，所以整体拓扑叫 **LCC-S**。

## 5. 一句话总结模型工作过程

**直流 15V → 全桥 Q1~Q4 逆变成高频方波 → 通过 LCC（Lf-Cf-Cp）补偿网络驱动发射线圈 Lp → 经磁耦合传到接收线圈 Ls → 由 Cs 串联补偿 → 二极管桥整流 → Cd 滤波后供给负载 RL。**

**附带运算部分**实时算 Pin、Pout、η（效率），并通过示波器观察 ILp、ILs、UCp、UCs、Uo 等关键波形来验证设计。
