# S-S 型无线电能传输（WPT）仿真模型 —— 中文标注说明

> 文件：`S_S.slx`（MATLAB 2020b 保存）
> 拓扑：原边串联（**S**eries）补偿 + 副边串联（**S**eries）补偿 → **S-S 拓扑**
> 应用：磁耦合谐振式无线充电系统（IPT/WPT）的最经典拓扑

---

## 1. 系统总体结构

整个仿真按照能量流方向，分为下面 6 大功率块 + 测量/运算辅助：

```
 ┌──────────┐  ┌─────────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
 │ 直流输入  │→│ 高频全桥逆变  │→│ 原边电容  │→│ 耦合线圈  │→│ 副边电容  │→│ 全桥整流  │→│ 滤波负载  │
 │  Vin=15V │  │  Q1~Q4      │  │   Cp     │  │ Lp ─ Ls  │  │   Cs     │  │ 二极管    │  │ Cd + RL  │
 └──────────┘  └─────────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘  └──────────┘
                                            电气主电路（功率回路）
                                                   │
                                                   ▼
                                       测量、PWM驱动、效率运算
```

S-S 拓扑特点：**原副边各只用一个串联电容**就能完成补偿，电路最简单，是 WPT 入门和理论分析最常见的拓扑。

> ⚠️ S-S 与 LCC-S 的对比
> - **LCC-S 原边**：Lf + Cf + Cp（3 个元件）
> - **S-S 原边**：仅 Cp（1 个元件）
> - **副边都一样**：只有 Cs

## 2. 元件 ↔ 中文名称 详细对照表

### 2.1 电源与逆变（原边输入级）

| 模型中名称 | 中文名称 | 作用说明 | 关键参数 |
|---|---|---|---|
| `DC Voltage Source1` | **直流输入电源 Uin（Vin）** | 给整个原边提供直流母线电压 | 幅值 = 15 V |
| `Mosfet4` (Q1) | **逆变开关管 Q1**（左桥臂上管） | 全桥逆变 4 个开关之一 | — |
| `Mosfet5` (Q2) | **逆变开关管 Q2**（右桥臂上管） | 与 Q1 互补 | — |
| `Mosfet6` (Q3) | **逆变开关管 Q3**（左桥臂下管） | 与 Q4 同时导通 | — |
| `Mosfet7` (Q4) | **逆变开关管 Q4**（右桥臂下管） | 与 Q1 同时导通（对角） | — |
| `Pulse Generator` | **PWM 驱动 1**（Q1/Q4） | 产生方波驱动对角开关管 | 高频方波 |
| `Pulse Generator1` | **PWM 驱动 2**（Q2/Q3） | 与 PWM1 反相 | 高频方波 |
| `Goto/From: Q1~Q4` | **PWM 驱动信号传递标签** | 把脉冲送到对应 MOSFET 的栅极 | — |

> 这 4 个 MOSFET + 2 路方波信号 = **高频全桥逆变器**，把直流变成方波交流去激励谐振网络。

---

### 2.2 S-S 补偿网络（核心）

| 模型中名称 | 中文名称 | 拓扑位置 | 参数 |
|---|---|---|---|
| `Series RLC Branch8` | **原边串联补偿电容 Cp** | 在桥臂中点和发射线圈 Lp 之间串联 | C = **84.52 nF**，R = 0.023 Ω（ESR） |
| `Mutual Inductance1` | **耦合线圈（Lp - Ls）** | 三绕组互感模块表示原副边的自感与互感 | Lp ≈ 30 µH，Ls ≈ 14 µH，M ≈ 5 µH |
| `Series RLC Branch5` | **副边串联补偿电容 Cs** | 与接收线圈 Ls 串联 | C = **181.11 nF**，R = 0.023 Ω（ESR） |

> S-S 的核心：**Cp 抵消 Lp 的感抗，Cs 抵消 Ls 的感抗**，使原副边电路在工作频率下都呈现纯阻性 → 功率因数 1，传输效率最高。
>
> 谐振条件：ω₀² = 1/(Lp·Cp) = 1/(Ls·Cs)
> 用你的参数验算：fp = 1/(2π·√(30e-6·84.52e-9)) ≈ **99.9 kHz**；fs = 1/(2π·√(14e-6·181.11e-9)) ≈ **99.9 kHz** ✅ 谐振点对齐良好。

| 模型中名称 | 中文名称 | 作用 | 参数 |
|---|---|---|---|
| `Universal Bridge1` | **副边全桥整流器** | 把高频交流转回直流 | 2 桥臂，二极管，Ron=1 mΩ |

## 3. 关键参数总览

| 参数 | 数值 | 说明 |
|---|---|---|
| 输入电压 Uin | 15 V | 直流源 |
| **原边补偿电容 Cp** | **84.52 nF** | 串联补偿（S） |
| 发射线圈 Lp | ~30 µH | 自感 |
| 接收线圈 Ls | ~14 µH | 自感 |
| 互感 M | ~5 µH | 耦合系数 k = M/√(Lp·Ls) ≈ 0.244 |
| **副边补偿电容 Cs** | **181.11 nF** | 串联补偿（S） |
| 输出滤波电容 Cd | 100 µF | 整流后滤波 |
| 负载电阻 RL | 10 Ω | 模拟负载 |
| 工作频率（理论） | ≈ 100 kHz | 由 LC 谐振决定 |
| MOSFET 导通电阻 | 0.044 Ω | — |

## 4. 一句话总结模型工作过程

**直流 15V → 全桥 Q1~Q4 逆变成 100 kHz 高频方波 → 经 Cp 串联补偿后驱动发射线圈 Lp → 通过磁耦合传到接收线圈 Ls → 经 Cs 串联补偿 → 二极管全桥整流 → Cd 滤波后输出到负载 RL。**

**辅助部分**实时计算 Pin、Pout、η（效率），并通过示波器观察 ILp、ILs、UCp、Uo 等关键波形。

## 5. 如何运行 `add_chinese_annotations_SS.m` 得到中文版模型

1. 把 `S_S.slx` 和 `add_chinese_annotations_SS.m` 放在**同一文件夹**；
2. 打开 MATLAB（≥ 2020b），用 `cd` 切到该文件夹；
3. 在命令窗口输入：
   ```matlab
   add_chinese_annotations_SS
   open_system('S_S_annotated')
   ```
4. 即可看到带 **蓝色中文注释 + 顶部总标题 + 红/绿色功能分区标题** 的中文版仿真模型 `S_S_annotated.slx`。

> 原始 `S_S.slx` **不会被改动**，所有标注只写到副本 `S_S_annotated.slx`。
